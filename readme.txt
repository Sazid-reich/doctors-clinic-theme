== Description ==
A clean, professional WordPress theme designed specifically for medical clinics and healthcare facilities.

== Features ==
* Clean medical-inspired design
* Fully responsive mobile-first layout
* Accessibility-ready
* Translation-ready
* Custom logo support
* Featured images
* Custom menus
* Threaded comments
* Custom colors
* Footer widgets
* Blog layout

== Installation ==
1. Upload the theme folder to /wp-content/themes/
2. Activate the theme in Appearance > Themes
3. Configure menu in Appearance > Menus
4. Add logo in Appearance > Customize > Site Identity

== Frequently Asked Questions ==
= Is this theme accessible? =
Yes, the theme follows WCAG 2.1 AA accessibility standards.

= Can I translate this theme? =
Yes, the theme is translation-ready with .pot file included.

= Does it work with page builders? =
Yes, the theme is compatible with all major page builders.
